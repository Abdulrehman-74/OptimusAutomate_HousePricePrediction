# House Price Prediction

## Machine Learning Internship Project

This project was completed as part of the Machine Learning Virtual Internship Program by Optimus Automate.

## Project Objective

The objective of this project is to predict house prices using machine learning techniques based on various housing features available in the dataset.

## Dataset

The dataset contains information about residential properties, including area, quality, number of rooms, year built, and other housing characteristics.

## Tasks Performed

* Data Loading and Exploration
* Missing Value Handling
* Categorical Data Encoding
* Outlier Handling
* Feature Selection
* Model Training using Random Forest Regressor
* Model Evaluation
* Data Visualization

## Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Machine Learning Model

* Random Forest Regressor

## Evaluation Metrics

The model was evaluated using:

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

## Visualizations

* Actual vs Predicted House Prices
* Feature Importance Analysis

## Project Outcome

The trained model successfully predicts house prices based on important housing features and demonstrates the application of machine learning techniques for regression problems.
